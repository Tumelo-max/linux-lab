# Linux & Networking Lab 1

This is my first Linux networking lab as part of my portfolio.

## Commands Run
1. `ping google.com` → tests connectivity to Google
2. `traceroute google.com` → shows packet route
3. `netstat -tulnp` → shows listening ports and processes
4. `ss -tulnp` → modern alternative to netstat

All outputs saved in `.txt` files.
