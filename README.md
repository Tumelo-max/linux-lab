
# Linux Networking & Cybersecurity Labs

This repository contains my Linux and networking lab exercises for my portfolio.

## Lab 1
- `ping google.com` → tests connectivity
- `traceroute google.com` → shows packet route
- `netstat -tulnp` → shows listening ports and processes
- `ss -tulnp` → modern alternative to netstat

## Lab 2
- `dig google.com` → DNS lookup
- `whois example.com` → domain info
- `top` → running processes & system stats
- `ps aux` → detailed process info
- `ping` & `traceroute` → network tests
- `netstat` & `ss` → network connections & listening ports
